# Elevatelab_Task_2

# Objective

The objective of this task is to perform data cleaning and handle missing values in a real-world dataset using Python. This task focuses on identifying missing data, visualizing missing value patterns, applying appropriate imputation techniques, and validating the cleaned dataset for further analysis or machine learning.


 # Dataset
 
- Name: House Prices Dataset
  
- Source: Provided as `house_prices.csv.zip`
   
- Format: CSV
  
- Description:The dataset contains housing-related attributes with both numerical and categorical features, including missing values that require preprocessing.


#  Tools & Libraries Used

- Python
  
- Pandas
   
- NumPy
   
- Matplotlib
    
- Google Colab  



# Steps Performed


- Identified missing values

- Visualized missing data
  
- Applied median imputation for numerical features
  
- Applied mode imputation for categorical features
  
- Removed columns with high missing values
  
- Validated cleaned dataset
  
- Compared dataset before and after cleaning


#  Results

- All missing values were successfully handled
  
- Dataset is clean and ready for further analysis or machine learning

- Data quality improved in terms of completeness and consistency


#  Outcome

The dataset was successfully cleaned and prepared for further analysis.
